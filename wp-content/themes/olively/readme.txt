== Description ==

Travel Blog WordPress Theme perfect for creating your own Travel Diaries. Document your exploits in style with Olively. A creative Front Page, Multiple Blog Layouts, Social Icons, Unique Design are some of the features of Olively. The clean design of the theme adds to the charm of the theme and helps you create an amzaing website. Supporting all the popular plugins, you can seek out help from the Support Forums. We are always ready to help. Hope you enjoy using Olively. 

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

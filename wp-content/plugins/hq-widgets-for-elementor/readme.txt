== Description ==
HQ Widgets for Elementor is a forever free plugin with a beautiful and intuitive widget for Elementor page builder.
- Designer Friendly
- Fully Responsive
- No Coding Required
- WooCommerce Compatible
- Translation Ready
- Use with Any Theme

* HQ Widgets for Elementor requires our plugin free plugin HQTheme Extra.

Some widgets work best with Marmot theme!
Note that you can turn the widget off to prevent unnecessary server load.

== Installation ==
Minimum Requirements

1. PHP 7.2 or greater is recommended
2. WordPress 5.5 or greater is recommended


Automatic installation
Automatic installation is the easiest option — WordPress will handles the file transfer, and you won’t need to leave your web browser. To do an automatic install of HQTheme Extra, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”
In the search field type “HQ Widgets for Elementor” then click “Search Plugins.” Once you’ve found us, you can view details. Click “Install Now,” and WordPress will take it from there.


Manual installation
Manual installation method requires downloading the HQTheme Extra plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains instructions on how to do this here.

Updating
Automatic updates should work smoothly, but we still recommend you back up your site.
